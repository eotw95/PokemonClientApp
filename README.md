# PokemonClientApp

PokemonClientApp uses PokeAPI to display the Pokemon Diagram and search for Pokemon. You can also create a Pokémon library of your favorite Pokémon.
PokeAPI https://pokeapi.co/

## Libraries

* Jetpack Compose
* Compose Navigation
* Coil
* Dagger Hilt
* LiveData
* Flow
* (Fragment) Navigation
* Jetpack Paging
* Room
* Retrofit
